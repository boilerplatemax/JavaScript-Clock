# 🕒 JavaScript + CSS Clock

An analog clock built using **vanilla JavaScript, HTML, and CSS**.  
The clock updates every second using the system time and rotates the clock hands dynamically using CSS transforms.

## 🚀 Features

- ⏰ Real-time clock powered by the JavaScript `Date` object  
- 🔄 Smooth rotation of hour, minute, and second hands  
- 🎨 Clean analog clock UI styled with CSS  
- ⚡ Automatic updates every second using `setInterval()`  
- 🧮 Accurate hand positioning using degree calculations  

---

## 🛠️ Built With

- **HTML5**
- **CSS3** (Transitions & Layout)
- **JavaScript** (Date API, DOM manipulation)

---

## 🧠 What I Practiced

- Working with the **JavaScript `Date` object**
- Converting time values into **rotation degrees**
- Dynamically updating element styles with `transform: rotate()`
- Using `setInterval()` for timed updates
- Handling the **CSS transition reset issue** when the second hand jumps from `59 → 0`
